Detailed Description

Class to write numpy matrices into Kaldi .ark file and create the corresponding .scp file.

It only supports binary-formatted .ark files. Text and compressed .ark files are not supported. The inspiration for this class came from pdnn toolkit (see licence at the top of this file)(https://github.com/yajiemiao/pdnn)
