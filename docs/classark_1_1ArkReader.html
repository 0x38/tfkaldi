Detailed Description

Class to read Kaldi ark format.

Each time, it reads one line of the .scp file and reads in the corresponding features into a numpy matrix. It only supports binary-formatted .ark files. Text and compressed .ark files are not supported. The inspiration for this class came from pdnn toolkit (see licence at the top of this file)(https://github.com/yajiemiao/pdnn)
